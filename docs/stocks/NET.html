About

Cloudflare is a software company based in San Francisco, California, that offers security and web performance offerings by utilizing a distributed, serverless content delivery network, or CDN. The firm's edge computing platform, Workers, builds on this infrastructure by enabling clients to deploy and run code without the need to manage or maintain servers.
